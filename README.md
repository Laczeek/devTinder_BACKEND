## Test it locally

-   Clonse the repository & Install the dependencies
-   Create .env with the content:
```
PORT=<VIRTUAL PORT>
MONGO_URL=<MONGO URL TO YOUR CLUSTER>
NODE_ENV=development
SECRET_KEY=<SECRET KEY FOR JWT>
```
-   Run npm run dev
